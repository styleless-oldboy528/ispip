## 🔍 如何检测IP纯净度？

购买代理IP后，建议先用专业工具检测IP纯净度，确保IP未被污染或列入黑名单，避免账号被封、平台限流等风险。

### ① 专业IP检测工具

> 📌 **推荐使用：https://ping0.cc/** （文章实测有效，可同时检测IP类型、风控值、原生状态）

将需要查询的IP输入，点击"查询"即可显示结果。主要看3个核心数据：

| 检测项目 | 合格标准 | 说明 |
|----------|----------|------|
| **IP类型** | ✅ 家庭宽带IP | 非机房/云服务器IP，真实住宅网络 |
| **风控值** | ✅ 极度纯净 | 值越小越安全，越不容易触发平台风控 |
| **原生IP** | ✅ 原生IP | 非广播/非转发IP，本地真实住宅IP |

> ⚠️ **重要提醒**：机房IP与住宅IP做业务都不是绝对的，只是相对而言，使用住宅IP更可靠、更安全、更不容易被风控。毕竟，**住宅IP是最纯净的IP类型**。

### ② 为什么使用原生住宅IP还是被风控？

一些用户反馈：为什么我使用了原生住宅IP还是被风控？可能原因：

- **设备指纹问题**：每台设备有唯一ID标识，一旦被拉黑，刷机重装也没用。建议使用指纹浏览器隔离环境
- **代理模式问题**：如果不清楚代理模式，建议切换成**全局代理模式**，确保所有流量走代理

住宅IP也叫原生IP，它们常用于：
- **跨境电商** — 访问本土电商平台、处理账户管理
- **社交网站注册** — 多账号矩阵运营，防关联
- **TikTok运营** — 直播、短视频、外贸运营

- **AI平台** — 访问海外AI服务

### ③ 住宅IP与机房IP的区别

| 类型 | 来源 | 特点 | 适用场景 |
|------|------|------|----------|
| **住宅IP** | 家庭宽带运营商（ISP） | ✅ 真实度高 ✅ 不易被封 ✅ 模拟真实用户 | 社媒运营、跨境电商、TikTok、账号注册 |
| **原生住宅IP（双ISP）** | 两家运营商双重认证 | ✅✅ 纯净度最高 ✅✅ 隐匿性最强 | 高端业务、TikTok直播、支付类平台 |
| **机房IP/云IP** | 数据中心或云服务商 | ✅ 稳定性高 ✅ 带宽大 | ❌ 易被识别 ❌ 容易被封 |

> 🧠 **选IP黄金法则**：越重要的业务，越需要纯净度高的住宅IP。TikTok直播、跨境电商账号建议直接上原生住宅IP双ISP，别省这点钱！

### ④ 静态住宅IP的特点

| 特点 | 说明 |
|------|------|
| 🔒 **安全稳定** | 由真实ISP提供，与真实家庭网络关联，具有更高的安全性和稳定性，有效避免IP被封禁 |
| 🌐 **全球覆盖** | 提供全球范围内的IP资源，覆盖多个国家和地区，满足全球业务需求 |
| ⚡ **高速连接** | 高带宽、低延迟的连接保证，让网络体验更加流畅，业务运行更加高效 |
| 🔄 **静态不变** | 固定不变的IP地址，适用于需要长期稳定IP环境的业务场景 |

> 💡 **静态住宅IP vs 动态住宅IP**：静态住宅IP是固定不变的IP地址，提供更稳定的连接和一致的身份标识，适合需要长期固定IP的业务；动态IP则适合需要频繁更换IP的场景。静态IP通常价格更高，但提供更可靠的服务质量。
